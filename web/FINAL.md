node server.js

## Ключевые особенности

### Значки
- Каждый репозиторий из Топ-500 имеет уникальный значок
- Значок показывает место в рейтинге
- Markdown разметка в колонке таблицы
- Клик по значку ведет на страницу репозитория

### Выделение строки
- При переходе по ссылке `?repo=123` строка выделяется
- Анимация привлекает внимание
- Автоматическая прокрутка к строке

### Цвета языков
- 11 известных языков (1C, C#, Java, Python и др.)
- 8 автоматических цветов для редких языков
- Консистентные цвета (один язык = один цвет)

### Компактная таблица
- Занимает 95% экрана
- Уменьшенные отступы
- Больше данных на экране
- Горизонтальная прокрутка

## Использование

### Для пользователей
1. Откройте http://localhost:8000
2. Перейдите в Топ-500
3. Найдите свой репозиторий
4. Скопируйте значок из колонки
5. Вставьте в README.md

### Для разработчиков
1. Данные загружаются с openyellow.org через прокси
2. Конфигурация в `js/config.js`
3. Легко переключить на API
