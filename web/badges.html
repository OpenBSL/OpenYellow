Значки проекта OpenYellow

Если вы - автор репозитория, который находится в ТОП-500, то вы можете помочь проекту OpenYellow, используя наш значок (badge) в своем репозитории. Он будет всегда показывать актуальную позицию вашего проекта в рейтинге, а при нажатии - перенаправлять на страницу ТОП-500 с фокусировкой соответствующей строке таблицы


Добавить такой значок себе очень просто - достаточно найти свой репозиторий в рейтинговой таблице и скопировать текст Markdown разметки в самой последней колонке. Этот текст, будучи вставленным в Readme файл репозитория, будет отображаться там как svg значок (как на картинке выше)
